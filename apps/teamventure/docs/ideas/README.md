## 使用说明

### 查看创意

1. **最新创意**: 点击上方"最新创意"章节的日期链接
2. **按月浏览**: 进入 `YYYY/MM/` 目录查看该月所有创意
3. **全文搜索**: 使用 IDE 或 `grep` 搜索关键词

### 手动触发

在项目根目录运行：
```bash
make generate-ideas
```

### 查看状态

```bash
make ideas-status
```

## 创意格式说明

每个创意包含以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| **标题** | 简明扼要的创意名称 | 方案对比智能排序 |
| **分类** | feature/performance/ux/architecture/security | ux |
| **优先级** | P0（最高）~ P3（最低） | P1 |
| **工作量** | S/M/L/XL | M（3-5天） |
| **描述** | 详细说明（包含背景、方案、实现要点） | 当前方案对比页按... |
| **预期收益** | 量化的用户价值或技术价值 | 提升点击率15% |
| **上下文** | 为什么现在提这个创意 | 根据 QA 报告... |
| **创意ID** | 唯一标识符（ULID格式） | idea_01kf7x... |

## 创意分类定义

### 🚀 功能增强（Feature）
新增功能或对现有功能的显著扩展，直接影响用户可用的能力。

**示例**：
- 增加新的活动类型（如密室逃脱、剧本杀）
- 支持导出方案为 PDF
- 添加方案收藏功能

### ⚡ 性能优化（Performance）
提升系统响应速度、降低资源消耗、提高并发能力。

**示例**：
- 地图图片预加载（减少加载时间）
- Redis 缓存优化
- 数据库查询优化

### 🎨 体验改进（UX）
改善用户交互流程、界面设计、易用性。

**示例**：
- 方案对比智能排序
- 增加引导提示
- 优化表单填写流程

### 🏗️ 架构优化（Architecture）
改进代码结构、降低耦合度、提升可维护性。

**示例**：
- 重构方案生成模块
- 引入事件溯源
- 优化分层架构

### 🔒 安全加固（Security）
增强系统安全性、数据隐私保护、防御能力。

**示例**：
- 添加 API 限流
- 敏感数据加密
- SQL 注入防护

## 优先级定义

| 优先级 | 定义 | 处理时间 | 示例 |
|-------|------|---------|------|
| **P0** | 影响核心功能或用户体验的严重问题 | 立即处理 | 支付失败、数据丢失 |
| **P1** | 重要功能改进，显著提升用户价值 | 1-2周内 | 智能排序、性能优化 |
| **P2** | 有价值但非紧急的改进 | 1个月内 | 界面美化、代码重构 |
| **P3** | 锦上添花的小功能 | 有空再做 | 彩蛋、实验性功能 |

---

## 工作量定义

| 工作量 | 时间 | 说明 | 示例 |
|-------|------|------|------|
| **S** | 1-2天 | 小改动，影响1-2个文件 | 添加字段、调整配置 |
| **M** | 3-5天 | 中等改动，跨多个模块 | 新增 API、缓存优化 |
| **L** | 1-2周 | 大功能，需设计和测试 | 新增活动类型、导出功能 |
| **XL** | 2周+ | 架构级改动，需详细规划 | 重构核心模块、迁移数据库 |
