# AI Coding Tutorial: 基于领域共同语言的稳态开发

> **核心理念**：AI 的不稳定性来源于上下文的模糊性。通过构建一套严密的“领域共同语言”和“契约文档”，我们可以将 AI 编程从“抽卡”变为“流水线生产”。

## 1. 项目简介
本项目是一个教学案例，旨在展示如何利用 LLM（大语言模型）进行高质量、可维护的软件开发。
我们将构建一个 **“企业活动预约系统” (Corporate Event Booking)** 的精简版。

### 为什么选择这个案例？
它包含了企业级应用的核心要素，但复杂度适中：
- **领域逻辑**：预约规则、库存扣减、状态流转。
- **多端协作**：管理后台 (Web) + 用户端 (小程序/H5)。
- **架构分层**：清晰的 DDD 分层架构。
