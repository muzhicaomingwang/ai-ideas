# Claude Code Dashboard

本地 Claude Code 使用数据可视化仪表盘，展示会话统计、Token 消耗、模型使用分布等指标。

## 快速开始

```bash
# 1. 生成/更新数据
./generate.sh

# 2. 在浏览器中打开
open index.html
```

## 功能特性

### 统计卡片
| 指标 | 说明 |
|------|------|
| 总会话数 | Claude Code 会话总数 |
| 总消息数 | 所有会话中的消息总数 |
| 最长会话 | 持续时间最长的单次会话消息数 |
| TOKEN消耗 | 所有模型的输入+输出 Token 总和 |
| SKILLS | 已配置的 Skills 数量 |
| 项目 | 项目目录数量 |

### 可视化图表
1. **每日活动趋势** - 折线图展示每日消息数变化
2. **Token消耗趋势** - 堆叠柱状图展示每日 Token 消耗（按模型分组）
3. **模型使用分布** - 环形图展示各模型 Token 占比
4. **活跃时段分布** - 柱状图展示 24 小时活动分布

## 数据来源

所有数据来自 `~/.claude/` 目录：

| 文件/目录 | 用途 |
|-----------|------|
| `stats-cache.json` | 主要统计数据（会话、消息、Token、活动分布） |
| `settings.json` | Skills 配置（用于计数） |
| `projects/` | 项目目录（用于计数） |

## 技术实现

- **纯静态页面**：无需服务器，双击即可打开
- **图表库**：Apache ECharts (CDN)
- **数据加载**：通过 Shell 脚本将数据内联到 HTML
- **深色主题**：护眼的深蓝黑色调

## 文件结构

```
dashboard/
├── generate.sh     # 数据生成脚本
├── index.html      # 仪表盘页面（生成）
└── README.md       # 本文档
```

## 更新数据

每次使用 Claude Code 后，运行以下命令更新仪表盘数据：

```bash
./generate.sh
```

脚本会读取最新的 `~/.claude/stats-cache.json` 并重新生成 `index.html`。
