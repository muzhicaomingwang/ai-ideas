## 单元0.1：课程介绍与学习方法（5分钟）

**讲师**：

“首先，我们来谈谈这门课独特的学习方法。它和传统的编程课非常不同，我们遵循一个四步循环：**观察、理解、验证和迭代**。

1.  **观察**：你需要仔细看我是如何向AI描述产品需求，以及AI是如何根据这些需求生成代码的。
2.  **理解**：你的重点不是记住代码长什么样，而是去理解每一段代码背后所实现的业务逻辑是什么。比如，这段代码是为了实现‘用户登录’，那段代码是为了‘显示活动列表’。
3.  **验证**：我们会把AI生成的代码，立刻在真实的环境中运行起来，检查功能是否和我们预想的一致。
4.  **迭代**：如果结果不完美，这很正常。你需要学习如何调整和优化你对需求的描述，引导AI给出更精准的答案。这就像与一位聪明的同事沟通，你需要清晰地表达你的想法。

那么，学习这门课，你‘**不需要**’做什么呢？

*   你**不需要**死记硬背任何编程语法。
*   你**不需要**理解复杂的技术底层原理。
*   你**不需要**有任何编程经验，零基础就是最好的起点。

你需要做的，恰恰是你最擅长的事情：

*   **清晰地描述产品需求**。
*   **验证功能是否符合预期**。
*   以及，**学会如何与AI高效沟通**。

好，作为本单元的第一个验证点，请大家思考一下：‘AI辅助开发’和‘传统编程学习’，这两者最核心的区别是什么？你可以在我们的课程交流群里分享你的看法。”

## 单元0.2：VPN与网络环境配置（5分钟）

**讲师**：

“接下来，我们进行第一个动手环节：配置网络环境。很多同学可能会问，为什么需要VPN？

原因很简单，我们使用的核心AI工具，比如Claude，它的服务器在国外。为了能够稳定、顺畅地访问这些服务，我们需要一个可靠的国际网络环境。同时，很多最新的开发文档、技术社区也都在国际网络上。

这里我推荐两种方案：

| 方案 | 适用场景 | 参考价格 |
|---|---|---|
| 机场服务 | 个人学习 | ¥20-50/月 |
| 企业VPN | 公司环境 | 咨询IT部门 |

对于个人学习者，‘机场服务’是性价比很高的选择。在配置时，请大家务必注意以下几点：

1.  选择‘**全局代理**’模式，而不是PAC模式，确保电脑上的所有网络请求都通过代理。
2.  配置完成后，请立刻访问 `https://claude.ai` 这个网址，如果能正常打开，就说明你的网络通了。
3.  同时，也要确保你的命令行工具（我们后面会讲）也能正常访问国际网络。

如果遇到连接不稳定的情况，可以尝试切换不同的节点，根据经验，日本或新加坡的节点通常速度会快一些。

**本单元的验证点**：请确保你能够成功并流畅地访问 `claude.ai` 网站。”

## 单元0.3：Git版本控制基础（5分钟）

**讲师**：

“我们继续下一个工具的准备：Git。你可以把Git想象成一个‘代码时光机’。

在开发过程中，我们的代码会不断修改。有了Git，你就可以在任何时候，让你的项目‘穿越’回任意一个历史版本，再也不用担心代码改错或丢失了。同时，它也是未来团队协作开发的基础。

我们只需要理解几个核心概念，不需要记忆命令，因为AI会帮我们处理。

| 概念 | 类比 | 说明 |
|---|---|---|
| Repository（仓库） | 项目文件夹 | 存放你所有代码和文件的地方 |
| Commit（提交） | 存档点 | 你为当前项目状态创建的一个快照或存档 |
| Branch（分支） | 平行世界 | 允许你同时开发多个功能而不互相干扰 |
| Pull/Push（拉/推） | 上传/下载 | 与云端的代码仓库同步你的本地代码 |

安装Git非常简单：

*   **Mac用户**：打开‘终端’，输入 `git --version`，如果系统提示你安装，按照指引操作即可。
*   **Windows用户**：请搜索并下载‘Git for Windows’进行安装。

在课程中，像 `git add`, `git commit` 这些常用命令，Claude Code都会自动帮我们执行，我们只需要理解它在做什么就行。

**本单元的验证点**：打开你的终端或命令行工具，输入 `git --version`，如果能看到版本号显示出来，就说明你安装成功了。”

## 单元0.4：Claude Code工具介绍（5分钟）

**讲师**：

“现在，让我们来认识一下本课程的绝对主角——Claude Code。

Claude Code是Anthropic公司开发的一款顶尖的AI编程助手。它的强大之处在于，能够深刻理解我们的自然语言需求，并直接生成高质量、可执行的完整代码。它就像一位资深的程序员坐在你旁边，你只需要告诉他‘做什么’，他就能帮你‘怎么做’。

它的核心能力包括：

1.  **代码生成**：这是我们最常用的功能，根据需求直接写代码。
2.  **代码解释**：如果你看不懂某段代码，可以让他为你解释。
3.  **问题修复**：当程序报错时，它可以分析错误并给出修复方案。
4.  **代码重构**：帮你优化现有的代码，使其更简洁、更高效。

课程中我们还会提到另一个工具Cursor，它和Claude Code有什么区别呢？

| 特性 | Claude Code | Cursor |
|---|---|---|
| 定位 | 终端AI助手 | IDE集成AI |
| 交互方式 | 命令行对话 | 编辑器内对话 |
| 适用场景 | 偏向于完成项目级的任务 | 偏向于在单个文件中进行编辑和修改 |
| 本课程角色 | **主力工具** | 辅助工具 |

简单来说，我们会用Claude Code来完成大部分从0到1的搭建工作，用Cursor来进行一些微调。

安装Claude Code需要一个叫做Node.js的环境。安装好Node.js后，在终端执行一行命令即可：`npm install -g @anthropic-ai/claude-code`。

**本单元的验证点**：安装完成后，在终端输入 `claude --version`，看到版本号即表示成功。”

## 单元0.5：微信小程序基础概念（5分钟）

**讲师**：

“我们的项目载体是微信小程序，所以我们需要对它有一个基本的了解。

小程序是运行在微信内部的一种轻量级应用，用户‘无需下载安装，即用即走’，非常适合工具类和服务类的产品。

我们只需要对它的技术架构有一个大概的印象。一个小程序项目，主要由一个全局配置文件 `app.json`，一些页面文件（比如首页 `index`），和一些工具函数 `utils` 组成。每个页面又由四种文件构成：

*   `.wxml` 文件：定义页面的**结构**，类似HTML。
*   `.wxss` 文件：定义页面的**样式**，类似CSS。
*   `.js` 文件：定义页面的**逻辑**，比如点击按钮后发生什么。
*   `.json` 文件：页面的独立配置。

大家完全不用担心，所有这些文件，AI都会帮我们生成和管理。

我们需要安装一个官方的**微信开发者工具**，它是我们预览、调试和发布小程序的官方IDE。大家可以从微信公众平台官网下载。

最后，有一个核心概念大家需要了解，就是‘本地存储’。你可以把它理解为小程序提供的一个‘本地数据库’，我们可以用它来保存一些简单的数据，比如用户的设置、导入的内容列表等。我们只需要知道有 `setStorageSync`（存数据）和 `getStorageSync`（取数据）这两个命令就可以了。

**本单元的验证点**：成功安装微信开发者工具，并能打开它创建一个测试项目。”

## 单元0.6：案例项目介绍 - TeamVenture（5分钟）

**讲师**：

“万事俱备，现在我们正式介绍本课程的实战项目——**TeamVenture**。

它的产品定位是：一款AI团建策划助手，旨在帮助公司的HR或行政人员，快速、智能地生成高质量的团建活动方案。

它的核心功能非常清晰：

1.  **需求收集**：用户输入团建的人数、预算、城市、以及风格偏好等。
2.  **方案生成**：AI根据需求，智能生成3套风格各异的团建方案，比如“刺激冒险型”、“休闲放松型”和“文化探索型”。
3.  **方案管理**：用户可以保存、对比、修改这些方案。
4.  **供应商对接**：对于最终确定的方案，可以一键联系合作的供应商（这部分在后续课程实现）。

所以，它的核心用户旅程就是：`输入需求 → AI生成方案 → 查看对比 → 确认方案`。

在本课程中，我们将重点完成前端小程序的核心功能开发。至于后端的服务和部署，我们会在后续的进阶课程中覆盖。

**本单元的验证点**：请大家花一点时间，完整地理解TeamVenture的产品定位和核心功能。想一想，如果你是用户，你期望它是什么样的？”
