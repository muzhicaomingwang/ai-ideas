# 背景音乐下载指南

为《完美的日子》播客准备免费的背景音乐（纯音乐版本）

---

## 推荐方案：使用 Pixabay 免费音乐库

**优势**：完全免费、无版权、高质量、无需注册

### 步骤1：访问 Pixabay 音乐库

打开浏览器访问：
```
https://pixabay.com/music/search/cinematic%20piano/
```

### 步骤2：筛选条件

- 点击 "Mood" → 选择 **Calm**、**Nostalgic**
- 点击 "Duration" → 选择 **2-5 minutes**
- 排序：**Most Popular**

### 步骤3：推荐曲目（选2首风格相近的）

搜索并下载以下关键词的音乐：

**第1首 - 替代《天堂电影院》**：
- 搜索: `nostalgic piano strings`
- 推荐: 带有弦乐的怀旧钢琴曲
- 下载后重命名为: `music/nostalgic-piano-1.mp3`

**第2首 - 替代《一个男人的梦想》**：
- 搜索: `calm piano meditation`
- 推荐: 宁静、深沉的钢琴独奏
- 下载后重命名为: `music/calm-piano-2.mp3`

## 备选方案：使用 Bensound

访问：
```
https://www.bensound.com/royalty-free-music/cinematic
```

推荐曲目：
- **Memories** - 怀旧钢琴，适合替代《天堂电影院》
- **Tomorrow** - 深沉钢琴，适合替代《一个男人的梦想》

**免费使用条件**：需在播客描述中注明来源

## 完成后执行混音

下载好2首音乐后，运行：

```bash
cd /Users/qitmac001395/workspace/QAL/ideas/apps/daily-podcast-ai

./venv/bin/python scripts/add_background_music.py \
  output/2026-01-08/podcast-perfect-days-review-final.mp3 \
  --music music/nostalgic-piano-1.mp3 music/calm-piano-2.mp3 \
  --output output/2026-01-08/podcast-perfect-days-with-bgm.mp3 \
  --volume 0.2 \
  --crossfade 5000
```

### 参数说明
- `--volume 0.2`: 背景音量为播客音量的20%
- `--crossfade 5000`: 两首音乐切换时5秒淡入淡出
- 脚本会自动循环播放这2首音乐，总共约播放2遍
